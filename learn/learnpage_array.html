Introduction to Array
Array overview:
The concept of an array is used in computer programming to describe a contiguous collection of items of the same type, with each element positioned relative to its base memory location to facilitate efficient storage and retrieval. An array simplifies calculations and operations by grouping similar data, allowing easy access to each element through a calculated offset. Providing a systematic approach to organizing and manipulating related data in a program, this fundamental data structure enhances code readability and efficiency.
Basic terminologies of array:
  • We can directly access an array element by using its index value.
    • Array Index:Array indexing starts with 0 so each element of an array is uniquely identified by its indexes.
    Array element: An individual item stored inside the array that may be retrieved by providing its index is known as the array element.
    Array Length: The longest possible array is the number of elements it can contain.

  • Definition:An array is a collection of elements, each of which is identified by an index.
